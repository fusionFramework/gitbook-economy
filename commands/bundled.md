# Bundled


### Pet
This type of action requires a pet to be selected.

| Name | Use |
| -- | -- |
| Feed | Changes hunger |
| Paint | Changes pet color (if possible) |
| Play | Changes mood
| Transform | Changes pet specie |

### User

| Name | Use |
| -- | -- |
| Avatar | Gives the user a locked avatar |
| Item | Gives out an item |
